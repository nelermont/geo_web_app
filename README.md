# Geo_web_app
Мультиплатформенное приложение для поиска Tile в Яндекс картах / Yandex map

Входнные данные - координаты и масштаб(зум)
На выходе получаем XY плитки и сам tile 

Стэк использован:
- Flutter/Dart
- Firebase/Firestore
